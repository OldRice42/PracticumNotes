How to Start Your Project and Submit It

You’ve learned the basics of the command line. You won’t have any problems moving from folder to folder. You can now work with Git: adding files to a commit, and pulling and pushing your and your team's work — everything you need for developing websites.

Now link your GitHub account to Practicum by Yandex so you can send in your projects for review. It’s easy. Go to your account and, in the navigation menu on the left, select the field with your project.

Click the Link GitHub Account button. You’ll be redirected to a browser so that our platform can collect your code. If you’re not signed in to GitHub, you’ll be asked to enter your username and password.
