HTML
Let's Start with HTML

Much like a Lego house is made of blocks, a webpage is assembled from elements, which are created and organized in HTML. Every page on the Internet has an HTML code and is saved as an HTML file, meaning that when you open a webpage in a browser, you are actually opening an .html file.

HTML is made up of tags, which can be recognized by their angle brackets. These tags serve as the building blocks that make up the framework of a webpage. They contain information about the role as well as the beginning and end of every single element. For example, with the help of tags, a browser can differentiate regular text from the text on a button.

CSS

Now we know about HTML — a language that is responsible for all the content on a web page. However, in order to change the color and the location of your content, you will have to use CSS. Without it, you wouldn't be able to make your text bright pink and place it in the center of the screen.

The description of an element's appearance is called style. CSS is the language of creating style.

A styles file has been linked to your project, which already contains some code. Your HTML code can be used to access content from this file. The styles file also contains some comments, which appear as gray text and are enclosed by certain symbols. The browser will ignore these comments, they're just there to give you some extra info.

Now let's take a look at this CSS code and start working on the recreation of the button.

Part 1

The styles have already been placed in the style.css file in a separate tab of the code editor. You can find this tab next to the index.html file. Go ahead and open it to take a look at the code. The styles are there, but they haven't yet been applied to the button.

Now we have to solve this issue. On the second line of the CSS file, we can see an element called buton. This means that the properties inside the curly braces will be applied to any element called buton in the HTML file.

Part 2

Now the button looks kind of old-school like it's from an old Nintendo game. This is cool, but it doesn't fit the style of our website. Our designer wants us to give it a more modern look.

JavaScript

You‘ve made the button look better than ever with CSS, but it still doesn't do anything. To turn your Like button into one that functions, you'll need to use Javascript. This is the language that brings page elements to life by making them interactive.

JavaScript code is often referred to simply as script, and script is what dictates how the browser works. Once the code starts running, the browser will perform every action that has been written in its script.

Take a look at line 13 in index.html and you will see that the script.js file is now linked to the page. Most of the code has already been written. All that's left is the fun part, which is to make your button clickable.

Part 1

The first step is to open the script.js file, which contains the script that controls the button.

The comments should make it easier for you to understand what’s going on in the code. What this is saying is that when someone clicks on the button, the script then finds the button and marks it as clicked.

ow let's get back to the button’s appearance. The script will add a clicked state to the button once it’s been clicked. However, this state does not yet have a style assigned to it. The clicked state should have a distinct style so that the button's appearance changes when someone clicks on it. That way, the user will know that the button works.

Let's make the clicked button black with white text. Do this by adding the following code in the last line of the style.css file, below the final closing brace }:

/* CSS */ .clicked { /* background-color */ background-color: black; /* color—for text color */ color: white; }
Part 2

Now your button works, but you can’t undo the like. Once the button has been clicked, it stays clicked forever and ever. Let’s give our users the ability to change their minds.

For this, you need to instruct the browser how to respond when a user clicks the button for the second time. The browser should remove the clicked state from the button and change the isClicked value to false. In order to do so, add the following code to line 10 in the script.js file:

//JavaScript button.classList = ''; isClicked = false;
